# Creating a New Branch and Merging Back to Main Branch

## Creating a New Branch

1. Click at the **bottom left corner**, where you see the **main branch**
2. The **Command Palette** will open and give you options to create a new branch or select an existing branch
3. From the Command Palette, click on **"Create new branch"**
4. Give the branch a name (e.g., "title-page", "font-style-change")
5. Check the **bottom left corner** to ensure you are now in your new branch

## Merging Back to the Main Branch

1. Click at the **bottom left corner**, where you see your **new branch name**
2. Select **"main"** from the Command Palette to switch back to the main branch
3. Open **Source Control**, click on the **three dots (...)**
4. Select **Branch â†’ Merge Branch...**
5. Select the **branch you want to merge** into the main branch
6. Click "Sync Changes"
7. The merge should now be completed

**Note:** Make sure to commit all changes in your feature branch before switching and merging!
